# UnityProjectDefaultFileGenerator

Unityでゲームを作る際に必要になるファイルを自動生成する.batです。

UnityのProjectの「Assets」ファイルの場所で「UnityProjectDefaultFileGenerator.bat」を
ダブルクリックで起動してください。

作成されるデータ
Project{
	Audeio,
	GameDate,
	Models,
	Prefab{
		System,
		UI,
		Date
	},
	Scenes{
		Debug,
		Main
	},
	Scripts{
		Core,
		InGamePlay,
		OutGamePlay,
	},
	UI
}
